# Perfect-Studio 
Perfect-Studio is a C++ application used in the process of manufacturing furniture.

# Introduction
This application lets you visualize the size, texture, and functionality of projected furniture. The program also provides an option to calculate the costs of manufacturing and exporting a list of needed assets. The application was written in C++ based on OpenGL and ImGUI.Â 

### Dependencies

* DirectX 11
* Latest ImGui version
* Windows compatible with 3.3 GLU Library
